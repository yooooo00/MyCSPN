# MyCSPN Project

## 项目概述

MyCSPN (My Convolutional Spatial Propagation Network) 是一个用于优化深度估计的效果神经网络。
项目代码修改自 [Convolutional Spatial Propagation Network](https://github.com/XinJCheng/CSPN)。
